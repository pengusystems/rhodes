Instructions for creating the vitis project on a clean copy from GIT
--------------------------------------------------------------------
1.Launch Vitis (from vivado or standalone)
2.Set workspace to this path
3.Close the welcome screen
4.Import from archive->Eclipse workspace or zip file
  1.root directory should be this path
  2.tick Search for nested projects
  3.untick Copy projects into workspace and everything else
5.project is now ready to be built
